Note

When reading tutorials from the internet, keep in mind that they might be written in Python 2. While most things work the same in both versions, there are some differences. One quick check is to look at the code, if it calls print without parentheses (print "foo"), then it is Python 2, in Python 3 the parentheses are required (print("foo")). Some of the key differences are explained, e.g. in a blog post by Sebastian Raschka.
